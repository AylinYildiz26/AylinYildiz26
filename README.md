### Selected work

**[Eniac — A/B Test Analysis](https://github.com/AylinYildiz26/Eniac_AB_Test_Analysis)**
Designed and ran the statistical evaluation of an e-commerce homepage CTA experiment (4 variants). Chi-square + Bonferroni-corrected post-hoc tests showed the "redesign" lost — the original button kept users from dropping off. Reads the experiment past the vanity metric to the decision that protects revenue.
`Python · SciPy · hypothesis testing`

**[Eniac × Magist — Strategic Data Audit](https://github.com/AylinYildiz26/Eniac_strategic_business_exploratory)**
SQL-based M&A due-diligence on a cross-border acquisition: exposed a "93% on-time" delivery claim as an artifact of padded estimates, and a two-speed logistics gap across regions. Delivered a conditional go/no-go with a phased integration roadmap.
`SQL · Tableau · commercial strategy`

**[GoExplore — Business KPI Dashboard](https://github.com/AylinYildiz26/GoExplore_business_kpi_dashboard)**
Built an end-to-end analytics setup from a single raw spreadsheet — BigQuery models feeding Looker dashboards — to answer market-expansion and channel-performance questions for a growing retailer.
`BigQuery · SQL · Looker Studio`

**[Liane's Library](https://github.com/AylinYildiz26/lianes-library)**
A full CRUD application (Python + MySQL + Streamlit) — relational schema, transactional create/update/delete, and an interactive front end. Built to show the engineering side, not just the notebooks.
